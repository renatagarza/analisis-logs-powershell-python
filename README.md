# Análisis automatizado de logs del sistema con PowerShell y Python

Este proyecto busca automatizar el proceso de recolección y análisis de logs del sistema Windows, utilizando PowerShell para extraer eventos y Python para procesar la información y generar reportes.

El objetivo es fortalecer las capacidades de monitoreo y detección de incidentes en entornos de ciberseguridad, aplicando buenas prácticas del análisis de registros.

## Tecnologías utilizadas
- PowerShell 5.1+
- Python 3.10+
- Librerías: pandas, matplotlib

## Estructura del proyecto
- `/src/powershell`: scripts para extracción de logs.
- `/src/python`: scripts para análisis de logs.
- `/outputs`: resultados generados.
- `/docs`: documentación técnica.
- `/proposals`: propuesta técnica inicial.
